This is cucumber java project done as part of Fancode interview process.

### System requirements
- JDK 11 or above


### Steps to clone and execute the tests

```
git clone https://github.com/ankitnandwani/FancodeChallange.git
```
```
mvn verify 
```


### Libraries Used
- Cucumber
- Rest Assured
- TestNg
- Owner
- Lombok
- Gson
